
# FormSettings element
Specifies source location and control settings for your mail add-in.

 **Add-in type:** Mail


## Syntax:


```XML
<FormSettings>
   ...
</FormSettings>
```


## Contained in:

[OfficeApp](../reference/manifest/officeapp-element.md)


## Can contain:

[Form](../reference/manifest/form-element.md)
